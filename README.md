# Worker-Connect

# 🛠️ Vizag Worker Connect

A hyperlocal service marketplace connecting users with nearby skilled workers like plumbers, electricians, and painters in Visakhapatnam.

---

## 🚀 Features

* 👤 User booking system
* 👷 Worker registration & availability
* 📍 Location-based worker discovery
* ⚡ Real-time booking updates
* ⭐ Ratings & reviews (future)

---

## 🧠 Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Mongoose)

### Real-time

* Socket.io

---

## 📦 Installation

### Clone the repo

```
git clone https://github.com/yourusername/vizag-worker-connect.git
```

### Backend setup

```
cd backend
npm install
npm run dev
```

### Frontend setup

```
cd frontend
npm install
npm run dev
```

---

## 🌍 Environment Variables

Create a `.env` file in backend:

```
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
GOOGLE_CLIENT_ID=your_google_client_id
```
